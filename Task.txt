Task

Написать программу для вычисления суммы чисел в файле

Дан json файл в виде массива объектов как показано ниже:
[
  {
    "a": 1,
    "b": 3
  },
  {
    "a": 5,
    "b": -9
  },
  {
    "a": -2,
    "b": 4
  }
  ...
]

Количество объектов = 1,000,000
значения чисел в диапазоне [-10,10] 


Программа должна считать файл и вычислить сумму всех чисел. Для этого она должна:
 1 Нужно распаралелить вычисление по горутинам
 2 Количество горутин, для параллельной обработки нужно получить при запуске программы через аргумент
 3 Вывести общий результат в консоль
 4 Опубликовать решение в github.com (http://github.com/) и предоставить доступ
